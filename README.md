# 🎉 EventCrewHub

**EventCrewHub** is a web-based platform that connects event managers with skilled crew members in real-time. It simplifies event staffing by replacing manual coordination with a centralized, efficient, and user-friendly system.

## 📖 About the Project

The event management industry often relies on manual methods like phone calls and messages for hiring crew members. This process is slow, unorganized, and inefficient.

**EventCrewHub** solves this problem by providing a digital platform where:

* Event managers can find and hire crew بسهولة
* Crew members can showcase their skills and availability
* Booking and communication happen in real-time

---

## ✨ Features

### 👤 User Features

* User Registration & Login (Manager / Crew)
* Profile Management (skills, experience, availability)
* Dashboard for tracking activities

### 🎯 Event Manager

* Create and manage events
* Search crew based on role & availability
* Send booking requests
* Approve / Reject applications

### 🧑‍🔧 Crew Member

* View available events
* Apply for events
* Accept / Reject bookings
* Track earnings & bookings

### 🔔 System Features

* Real-time notifications
* Secure authentication & data handling
* Role-based access control

## ⚙️ Installation

### Prerequisites

* Java JDK 8+
* Apache Tomcat (v9+)
* MongoDB (Community Server)
* VS Code / Eclipse

### Steps

1. Clone the repository

```bash
git clone https://github.com/your-username/EventCrewHub.git
```

2. Open project in IDE (VS Code / Eclipse)

3. Configure MongoDB connection

4. Deploy project on Apache Tomcat

5. Run the server:

```bash
http://localhost:8080/EventCrewHub
```

## ▶️ Usage

1. Register as **Event Manager** or **Crew Member**
2. Login to the system
3. Managers → Create events & hire crew
4. Crew → Apply for events
5. Manage bookings and track status
